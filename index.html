Apender HTML

¿Que es HTML?

HTML significa "HyperText Markup Language" (Lenguaje de Marcado de Hipertexto), es el lenguaje de marcado estándar utilizado para crear páginas web. Fue creado con el propósito de estructurar y presentar el contenido en la web de una manera organizada y coherente, permitiendo a los navegadores web interpretar y mostrar el contenido correctamente.


HTML utiliza etiquetas y elementos para definir la estructura de una página web. Cada etiqueta proporciona información al navegador sobre cómo debe mostrar y renderizar el contenido.

Historia de Html

El HTML nace en 1980 cuando el científico Tim Berners-Lee propuso un nuevo sistema de hipertexto para compartir documentos. Surge ante la necesidad de encontrar documentos que te aporten información relacionada con el tema que necesitas y que, además, puedas acceder a ellos desde cualquier parte del mundo.


Es por ello por lo que un grupo de expertos con Berners-Lee de cabecilla deciden crear un nuevo proyecto: el World Wide Web (WWW). Una de las bases de este proyecto es el lenguaje HTML, que como ya hemos mencionado permitirá que cada uno de los documentos que están en internet tengan un formato unificado y que le dé una estructura.


La forma de hacerlo es a través de etiquetas con un orden determinado. Recordemos que este lenguaje sólo podrá ser procesado por un navegador como los que usamos a diario

Versiones

  1. Principios de la década de 1990: Berners-Lee desarrolló el HTML como una forma de estructurar la información en la web. En 1993, se publicó la primera especificación HTML llamada "HTML Tags", que proporcionaba una forma básica de marcar la estructura del contenido.

  2. HTML 2.0 (1995): El primer estándar formalizado de HTML fue HTML 2.0, que introdujo elementos como las imágenes y los formularios.

  3. HTML 3.2 (1997): Esta versión agregó tablas y marcos, lo que permitió un mayor control sobre la presentación de la página.

  4. HTML 4.0 (1997-1999): HTML 4.0 introdujo más elementos y atributos, como capas y hojas de estilo en línea. También marcó la separación entre la estructura (HTML) y la presentación (CSS) de las páginas web.

  5. HTML5 (2008-2014): TML5 fue un cambio importante en la historia de HTML. Introdujo muchas nuevas características, como elementos semánticos (encabezados, secciones, artículos, etc.), elementos multimedia (audio y video), el elemento "Canvas" para gráficos, y una API más rica para interactividad. También permitió la incorporación de elementos personalizados y facilitó la creación de aplicaciones web más ricas.

  6. HTML Living Standard (2012-presente): En lugar de versiones discretas, HTML ahora sigue una especificación viva llamada "HTML Living Standard". Esto significa que el estándar se sigue actualizando y evolucionando continuamente.

  7. Web Components y HTML6 (futuro): A medida que la web continúa desarrollándose, se están explorando conceptos como los Web Components para crear elementos web personalizados y reutilizables. Aunque se habló de HTML6, la idea de una versión independiente de HTML después de HTML5 parece haber sido reemplazada por la noción de la especificación "HTML Living Standard".

Etiquetas en HTML

Son elementos fundamentales utilizados para estructurar y marcar el contenido en una página web. Las etiquetas son instrucciones que le dicen al navegador cómo mostrar y organizar los diferentes elementos en una página. Cada etiqueta se compone de una etiqueta de apertura, contenido y, en algunos casos, una etiqueta de cierre. Juntas, las etiquetas forman el "marcado" que define la estructura y el formato de una página web.

Las etiquetas se delimitan usando paréntesis angulares (“< >”) y se utilizan para describir algo que se quiere representar en una página web, su apariencia.

Tipos de etiquetas

  1. Etiqueta !DOCTYPE: Se utiliza para especificar la versión de HTML que se está utilizando en la página.

  2. Etiqueta html: Define el inicio y el final de la página web.

  3. Etiquetas de apertura: Son el inicio. Indican dónde comienza el elemento o donde comienza a tener efecto. Consisten en el nombre del elemento encerrado entre los corchetes angulares. '<>'

  4. Etiquetas de cierre: Son lo mismo que las etiquetas de apertura, pero estas indican donde finaliza el elemento. Además, se diferencian de las otras ya que antes de escribir el elemento entre los corchetes, debemos escribir una barra diagonal. "/"

  5. Etiqueta de sección superior: head Esta etiqueta indica la parte superior del texto o encabezado.

  1. Etiquetas de título: title Se usa para indicar que lo que se pondrá a continuación es el título de la página web.

  2. Etiqueta de título: h1 Título o encabezado de la página.

  3. Etiqueta de subtítulo: h2 Título de nivel 2 o subtítulo.

  4. Etiqueta de apartado: h3 Subapartado. También existen h3, h4, h5 , h6.

  5. Etiquetas de párrafo: p Hace que cada elemento o texto aparezca en una nueva línea agrupado.

  6. Etiquetas de imagen: img Se emplea para insertar una imagen en la página en el lugar donde se quiere incluir. Esta etiqueta no necesita una de cierre.

  7. Etiquetas de enlace: a Permite generar un vínculo o link.

Estructura de HTML

La estructura básica de un documento HTML consta de varias etiquetas que definen la estructura y el contenido de una página web. Aquí tienes un ejemplo de la estructura HTML típica:


  1. <!DOCTYPE html>: Como ya antes explicamos nos permite definir que utilizaremos la ultima version de html.
  2. <html lang="es">: Es el elemento raíz que contiene todo el contenido de la página.
  3. <head>: Contiene información meta, enlaces a estilos y otros elementos que no se muestran directamente en la página.
  4. <meta charset="UTF-8">: Especifica la codificación de caracteres del documento.
  5. <meta name="viewport" content="width=device-width, initial-scale=1.0">: Ajusta la escala y el ancho de la página para dispositivos móviles.
  6. <title>: Define el título de la página que se muestra en la pestaña del navegador.
  7. <title>: Define el título de la página que se muestra en la pestaña del navegador.
  8. <body>: Contiene el contenido visible de la página.
  9. <header>: Contiene elementos de encabezado, como títulos y menús de navegación.
  10. <nav>: Define un área de navegación.
  11. <main>: Contiene el contenido principal de la página.
  12. <seccion>: Define secciones de contenido.
  13. <footer>: Contiene información del pie de página.

Ejemplo del codigo HTML


  <!DOCTYPE html>
  <html lang="es"> <!-- lang define el idioma de la pagina -->
  <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, 
          initial-scale=1.0">
          
      <title>Título de la Página</title>
  </head>
  <body>
      <header>
          <h1>Encabezado de la Página</h1>
          <nav>
              <!-- Menú de navegación -->
          </nav>
      </header>
              
      <main>
          <section>
              <h2>Sección 1</h2>
              <!-- Contenido de la sección -->
          </section>
              
          <section>
              <h2>Sección 2</h2>
              <!-- Contenido de la sección -->
          </section>
      </main>
              
      <footer>
          <p>Pie de Página</p>
      </footer>
  </body>
  </html>

Formularios en HTML

Parte fundamental de las paginas webs son los formularios, un formulario en HTML es un elemento que se utiliza para recopilar información o datos de los usuarios que visitan un sitio web. Los formularios permiten a los usuarios ingresar datos, seleccionar opciones y enviar esa información al servidor web para su procesamiento. Los formularios son una parte fundamental de muchas páginas web, ya que permiten la interacción entre los usuarios y la aplicación web.
Un formulario en HTML está compuesto por varios elementos:

Elemento <Form>

Este elemento es el contenedor principal del formulario. Contiene todos los elementos relacionados con el formulario, como campos de entrada, botones y etiquetas.

Elemento <Input>

Es uno de los componentes más versátiles para crear campos de entrada en formularios. Se utiliza para recopilar diferentes tipos de datos ingresados por los usuarios. Algunas de las propiedades más comunes que puedes usar con el elemento son


Type: Define el tipo de campo de entrada que se va a mostrar. A continuacion algunos de los mas valores comunes

Text

Campo de texto de una sola línea.

Password

Campo de contraseña, los caracteres se ocultan.

Email

Campo de entrada para direcciones de correo electrónico.

Number

Campo de entrada numérica.

Checkbox

Campo de entrada numérica.

Radio

Botón de radio para selección única.

Submit

Botón de envío para enviar el formulario.

Reset

Botón para restablecer el formulario.

File

Permite a los usuarios cargar archivos desde su dispositivo.

Date

Muestra un campo de entrada para seleccionar una fecha.

textarea: Se utiliza para crear un área de texto multilínea en formularios web. A diferencia del elemento <input> de tipo "text", que solo permite una línea de texto, <textarea> permite a los usuarios ingresar texto en varias líneas, lo que lo hace ideal para recopilar comentarios, descripciones largas u otros tipos de contenido textual más extenso.

Etiqueta label: Se Se utilizan para proporcionar una descripción o nombre para cada campo de entrada. Al hacer clic en la etiqueta, se activa o selecciona el campo correspondiente.

Tablas en HTML

Son elementos utilizados para organizar datos en filas y columnas. Permiten presentar información en un formato tabular, lo que facilita la visualización y comprensión de datos estructurados. Las tablas constan de varias partes, incluyendo filas, columnas y celdas.

Una tabla se define con la etiqueta <table> y dentro de ella <tr> el cual define las filas dentro de cada fila utilizamos <th> para encabezados de la columnas y la etiqueta <td> para celdas de datos.

Las tablas en HTML son útiles para presentar datos tabulares como listas, horarios, resultados y otros tipos de información organizada. Sin embargo, para diseño de página y maquetación, se recomienda el uso de CSS y elementos semánticos como <div> y <section>.

JavaScript y CSS

CSS y JavaScript desempeñan papeles cruciales en el desarrollo web moderno. CSS se encarga de la presentación y el estilo de un sitio web, mientras que JavaScript proporciona interacción dinámica y funcionalidad avanzada. Ambos lenguajes trabajan en conjunto para crear experiencias web atractivas y efectivas para los usuarios.


CSS se utiliza para controlar la presentación y el estilo de una página web. Esto incluye la apariencia visual, como colores, fuentes, márgenes, alineación y diseño, CSS permite separar el contenido (HTML) de la presentación. Esto facilita la modificación del aspecto de un sitio web sin cambiar su estructura subyacente, lo que mejora la mantenibilidad.


JavaScript es un lenguaje de programación que se utiliza para crear interacciones dinámicas en una página web. Permite a los usuarios realizar acciones como enviar formularios, realizar búsquedas, desplazarse por páginas y más, todo sin necesidad de recargar la página.
